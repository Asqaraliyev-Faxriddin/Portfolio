Men haqimda

Professional faoliyatimning boshida bo‘lishimga qaramay, ko‘plab Telegram botlar va full-stack veb loyihalarni ishlab chiqqanman. Backend arxitekturalarini loyihalash, ma’lumotlar bazalari (SQL) bilan ishlash va samarali algoritmlar yaratishda tajribaga egaman. Loyiha ishlarim ko‘pincha admin va foydalanuvchi panellari, rollarga asoslangan kirish nazorati (RBAC) hamda murakkab mantiqiy yechimlarni o‘z ichiga oladi.

Najot Ta’limda Full Stack kursining backend modulini muvaffaqiyatli tamomlaganman va hozir frontend modulida bilimlarimni oshirmoqdaman. Bu davr mobaynida REST va GraphQL API larini yaratish, foydalanuvchilarni autentifikatsiya va avtorizatsiya qilish, JWT va refresh tokenlar orqali xavfsizlikni ta’minlash, Clean Architecture va MVC kabi testlanuvchi arxitekturalarni qo‘llashda amaliy ko‘nikmalarga ega bo‘ldim.

Shuningdek, backend xizmatlari bilan integratsiyalangan Telegram botlar ishlab chiqdim, bu foydalanuvchilar uchun avtomatlashtirilgan va interaktiv funksiyalarni taqdim etdi. Asinxron amallar, tranzaksiyalar, ACID prinsiplari orqali ma’lumotlar ishonchliligi va xavfsizligini ta’minlashni o‘rgandim.

Hozirda React.js, JavaScript va Tailwind CSS yordamida zamonaviy, responsiv va foydalanuvchilarga qulay interfeyslar yaratishni o‘rganmoqdaman. Backend va Frontend sohalaridagi qo‘shimcha tajribam tufayli men yuqori sifatli, to‘liq ishlaydigan veb-ilovalarni yaratishga qodir bo‘lgan to‘laqonli Full Stack dasturchiga aylanib boryapman.

Texnologiyalar: Node.js, Express.js, Nest.js, GraphQL, REST, Socket.io, PostgreSQL, MySQL, MongoDB, Redis, Mongoose, Prisma, Sequelize, Docker, Jest, CI/CD, PassportJS, Microservices, RabbitMQ, AWS Cloud va boshqalar.
